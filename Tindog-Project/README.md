# TinDog Landing Page

This project is a static landing page designed for a fictional application named "TinDog". It was developed to practice responsive design principles and the Bootstrap framework as part of The Complete 2024 Web Development Bootcamp by Angela Yu.

## Project Overview
The main objective of this project was to build a professional-looking landing page without using any backend functionality. The focus is entirely on the frontend user interface, layout structure, and responsiveness across different device sizes.

## Technologies Used
* HTML5
* CSS3
* Bootstrap 4
* FontAwesome
* Google Fonts (Montserrat & Ubuntu)

## Key Learnings
During the development of this project, the following technical concepts were applied:

* Bootstrap Grid System: Utilized col-lg and col-md classes to create a responsive layout that adapts to mobile, tablet, and desktop screens.
* CSS Positioning: Applied z-index and absolute positioning to manage overlapping elements.
* Media Queries: Implemented custom CSS breakpoints to handle specific design requirements on smaller screens.
* File Path Management: Practiced debugging and organizing asset directories correctly.
